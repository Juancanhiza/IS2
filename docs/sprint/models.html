metodo del modelo Sprint que retorna la cantidad de dias de duracion del sprint en dÃ­as hÃ¡biles :return: :dias: la cantidad de dias entre la fecha de inicio del sprint y la fecha de finalizacion del sprint, en caso de no tener fecha de finalizacion aun, hasta la fecha de hoy

    def get_duracion_real(self):
# 
        if not self.fecha_inicio: return 0
        dias_habiles = self.get_dias_habiles()
        inicio = self.fecha_inicio
        if self.fecha_fin:
            fin = self.fecha_fin
        else:
            fin = timezone.now().date()
        currentdate = inicio
        dias = 0
        while currentdate <= fin:
            if currentdate.isoweekday() in dias_habiles:
                dias += 1
            currentdate += timedelta(days=1)
        return dias
